The Project abides by our \[code of conduct](./CODE\_OF\_CONDUCT.md).



\## How To Contribute Code To Our Project 



\### Setting Up Your Environment



Fork the Project repository on Github and clone the fork onto your local machine.



```cmd

git clone https://github.com/YOUR-USERNAME/bank\_marketing\_group\_24.git

```



Create the virtual environment our team uses with the provided environment.yml file



```cmd

conda create -f environment.yml

```



Once the environment is solved, activate the environment to start using it



```cmd

conda activate 552

```



\### Creating a Branch



Create a new Github branch that will contain your new contributions



```cmd

git switch -c <branch-name>

```



Be sure to regularly test your code to ensure compatibility with the source code written by our team.



\### Publishing Your Changes



Once you are happy with your changes, commit and push them to the remote repository by performing the following commands:



```cmd

git add <modified-file>

git commit -m "A description of the changes you made"

git push origin <branch-name>

```



Next, create a pull request on Github and ask one of our team members to review your changes for them to be merged to the main branch.
